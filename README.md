# Montasir Moyen

> [!NOTE]  
> Live at: https://www.montasirmoyen.com/

## Personal Website

### Built with
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion

### Contains

- **Home** (`/`) — Landing page with animated intro, tech stack, bento grid, and featured projects.
- **About** (`/about`) — Background, skills, and personal info.
- **Projects** (`/projects`) — List of projects with summaries.
- **Project Detail** (`/projects/[slug]`) — Individual project page.
- **Blog** (`/blog`) — Blog post listing.
- **Blog Post** (`/blog/[slug]`) — Individual blog post.
